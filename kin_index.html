How to Use for Clinical Assessment

  1. Position patient 2–4 metres from camera, full body visible
  2. Ensure good lighting — avoid strong backlighting
  3. Ask patient to perform target movement (gait, squat, lunge, etc.)
  4. Monitor live angle readouts and quality score
  5. Download CSV for session documentation
  6. Compare sessions over time for treatment outcome evaluation
⚠️ Clinical Disclaimer: KinetIQ provides movement data to support — not replace — clinical judgement. All findings should be interpreted by a qualified physiotherapist or medical professional. Not validated as a medical device.

AI Pose Engine
MediaPipe Pose detects 33 body landmarks at up to 30 fps. Joint angles are computed from landmark coordinates using vector mathematics.
🔒
100% Private
All processing happens on your device. No video, images, or patient data ever leave your browser. HIPAA / GDPR compatible deployment.
📱
Universal Device Support
Tested on Chrome, Firefox, Safari, Samsung Internet, Xiaomi MIUI Browser, and Huawei Browser. Works on Android, iOS, Windows, macOS, and Linux.

Week 6 Colab Integration
Export session CSV and import directly into the companion Google Colab notebook for advanced statistical analysis, pre/post treatment comparison, and publication-ready figures.
🌐
Deploy Anywhere
Single HTML file — host on GitHub Pages, Netlify, hospital intranet, or any static file server. No backend, database, or installation required.
📐
Angle Algorithm
Joint angles computed as the angle between two vectors sharing a common vertex (joint centre). Hip, knee, and ankle angles derived from MediaPipe 3D landmark coordinates.
